# 问题描述
&emsp;给出一个字符串$S$, 手动实现atof, 将S中包含的所有数字输出, 保留6位小数.   
&emsp;两个浮点数之间会用空格隔开, 可能存在负数.  
# Sample Input
```
123.456 0.3 -12.3 23 0.12 12. .2 -.2
```
# Sample Output
```
123.456000 
0.300000
-12.300000
23.000000
0.120000
12.000000
0.200000
-0.200000
```
